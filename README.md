# لوحة متابعة تجهيز الفرع — قوين | Qween Branch Dashboard

لوحة تحكم عربية (RTL) لمتابعة تسجيل وتعبئة أقسام الفرع الجديد بالمجموعات الفرعية والموردين — تعمل على الجوال واللابتوب بدون إنترنت وبدون أي تثبيت.

**🔗 الرابط المباشر:** https://gzjz10.github.io/qween-branch-dashboard/

## المميزات

- **نظرة عامة**: مؤشرات الفرع (المستهدف، المسجل، العجز، نسبة الإنجاز، التجاوز، الفواتير والمبالغ المسددة) وجدول ملخص لكل الأقسام.
- **18 قسمًا**: المنظفات، الإسفنجيات، المناديل المبللة، المكياج، العناية بالبشرة، الألعاب، الورود، السبالة، الغذائية، المكتبية، الملابس الداخلية، البلاستيك الأرضي، الاستهلاكية، الفحم، المفروشات، الإكسسوارات، العطور، الحفلات.
- **جدول الأصناف الفرعية** لكل قسم: الكمية لكل مورد، الإجمالي، العجز، نسبة الإنجاز، والحالة تلقائيًا:
  - ✅ **مكتمل** ≥ 90%
  - 🟡 **تحت التنفيذ** ≥ 60%
  - 🟠 **يحتاج متابعة** ≥ 30%
  - 🔴 **لم يبدأ** أقل من ذلك
  - 🟣 **تجاوز المستهدف +N** — عند الشراء أكثر من المستهدف (ميزة إضافية غير موجودة في ملف الإكسل الأصلي)
- **تحرير مباشر**: عدّل الكميات والمستهدفات والملاحظات وبيانات القسم — تُحسب الإجماليات فورًا وتُحفظ تلقائيًا في المتصفح.
- **تصدير واستيراد**: نسخة احتياطية JSON، وتصدير CSV يفتح في Excel بالعربية بشكل صحيح.
- **بحث وتصفية**: بحث عن أي صنف فرعي وتصفية حسب الحالة.
- **متجاوب**: جداول كاملة على اللابتوب، وبطاقات قابلة للطي على الجوال.

## الاستخدام

- **أونلاين**: افتح الرابط أعلاه من أي جهاز.
- **بدون إنترنت**: حمّل `index.html` وافتحه مباشرة — الملف مستقل تمامًا.

> **ملاحظة**: البيانات تُحفظ محليًا في متصفح كل مستخدم (localStorage). لمشاركة التعديلات بين الموظفين استخدم زر «تصدير JSON» وأرسل الملف، ثم «استيراد JSON» عند الطرف الآخر.

## الملفات

| الملف | الوصف |
|---|---|
| `index.html` | التطبيق كاملًا (بيانات + واجهة + منطق) في ملف واحد |
| `data.json` | البيانات الأصلية المستخرجة من ملف الإكسل (نسخة مرجعية) |
| `SPEC.md` | المواصفات الفنية الكاملة |
| `CLAUDE.md` | إرشادات التطوير لمن يعدّل المشروع لاحقًا |
