# لوحة متابعة تجهيز الفرع — قوين | Qween Branch Dashboard

لوحة تحكم عربية (RTL) لمتابعة تجهيز الفرع الجديد **مع نظام متكامل لأوردرات الموردين** — تعمل على الجوال واللابتوب بدون إنترنت وبدون أي تثبيت، في ملف واحد.

**🔗 الرابط المباشر:** https://gzjz10.github.io/qween-branch-dashboard/

## المميزات

- **نظرة عامة**: مؤشرات الفرع (المستهدف، المسجل، العجز، نسبة الإنجاز، التجاوز، الفواتير والمبالغ المسددة)، جدول ملخص لكل الأقسام، ورسم بياني لحالة الأوردرات وأعلى الموردين (عند وجود أوردرات).
- **21 قسمًا «قطاعي»**: ملحقات المنزل/المطبخ/المائدة، المنظفات، أدوات التجميل، البلاستيكات، الإكسسوارات، القرطاسية، المفروشات، كهربائيات وخردوات، السفر والأكياس، الألعاب، الملابس، العطور، العناية بالبشرة، الحفلات، المواد الغذائية، المياه، المستلزمات الرمضانية، الورقيات والمناديل، التحف والهدايا.
- **جدول الأصناف الفرعية** لكل قسم: الكمية لكل مورد، الإجمالي، العجز، نسبة الإنجاز، والحالة تلقائيًا:
  - ✅ **مكتمل** ≥ 90%
  - 🟡 **تحت التنفيذ** ≥ 60%
  - 🟠 **يحتاج متابعة** ≥ 30%
  - 🔴 **لم يبدأ** أقل من ذلك
  - 🟣 **تجاوز المستهدف +N** — عند الشراء أكثر من المستهدف (ميزة إضافية غير موجودة في ملف الإكسل الأصلي)
- **الموردون**: إضافة وتعديل الموردين، ربط كل مورد بالأقسام التي يورّدها، بحث وتصفية.
- **الأوردرات**: معالج من 4 خطوات لإنشاء أمر (اختيار المورد → الأقسام → الكميات لكل صنف → المراجعة)، مع تعديل الأمر، وحالات (مسودة، قيد الانتظار، مؤكد، تم التسليم، ملغي). **عند «تم التسليم» تُرحَّل الكميات تلقائيًا إلى اللوحة**، وتُسحب تمامًا عند التراجع.
- **متابعة الموردين**: تجميع أداء كل مورد عبر أوامره مع تفصيل حسب القسم.
- **التقارير**: تقارير الموردين والأوردرات والمجموعات والعجز — قابلة للطباعة (تُنسّق العربية بشكل صحيح).
- **تحرير مباشر**: عدّل الكميات والمستهدفات والملاحظات وبيانات القسم — تُحسب الإجماليات فورًا وتُحفظ تلقائيًا في المتصفح.
- **تصدير واستيراد**: نسخة احتياطية JSON كاملة، وتصدير CSV (الأصناف، الموردون، الأوردرات) يفتح في Excel بالعربية بشكل صحيح، وبيانات تجريبية للتجربة.
- **بحث وتصفية**: في كل شاشة، وتصفية حسب الحالة.
- **متجاوب**: جداول كاملة على اللابتوب، وبطاقات قابلة للطي على الجوال.

## الاستخدام

- **أونلاين**: افتح الرابط أعلاه من أي جهاز.
- **بدون إنترنت**: حمّل `index.html` وافتحه مباشرة — الملف مستقل تمامًا.

> **ملاحظة**: البيانات تُحفظ محليًا في متصفح كل مستخدم (localStorage). لمشاركة التعديلات بين الموظفين استخدم زر «تصدير JSON» وأرسل الملف، ثم «استيراد JSON» عند الطرف الآخر.

## الملفات
